# My Portfolio

Static portfolio website for RohithKumar Saravanan.

## Preview
Open `index.html` in a browser or serve it from a local web server.

## Project structure
- `index.html` — main portfolio page
- `assets/` — image and icon assets used by the site

## Notes
This is a static HTML/CSS/JS site using Tailwind via CDN and GSAP for scroll animations.
